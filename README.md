# IoT-Based Smart Helmet with Accident Detection and Location Tracking

## Overview
This project is an IoT-based smart helmet designed to improve rider safety. It detects accidents using an MPU6050 sensor, obtains the accident location using a GPS module, and sends an emergency SMS through a GSM module.

## Features
- Accident Detection
- GPS Location Tracking
- GSM Emergency SMS Alert
- LCD Display
- Buzzer and LED Indication
- Emergency Push Button

## Hardware Components
- Arduino Mega 2560
- MPU6050 Sensor
- GPS Module (NEO-6M)
- SIM900A GSM Module
- 16×2 LCD Display
- Push Button
- LEDs
- Buzzer

## Software
- Arduino IDE
- Embedded C/C++

## Working
1. The MPU6050 continuously monitors helmet movement.
2. If an accident is detected, the GPS module reads the location.
3. The SIM900A sends an emergency SMS containing the GPS location.
4. The LCD displays the system status.
5. The buzzer and LEDs alert nearby people.

## Applications
- Motorcycle Safety
- Emergency Response
- Accident Monitoring
